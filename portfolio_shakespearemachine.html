Shakespeare Machine

Shakespeare Machine is a large-scale multimedia sculpture shows words extracted from Shakespeare’s original masterpieces.
· Indoor sculpture    · Art Installation    · Big Data  
Project Details

Client: The Public Theater

Date: 2012

Online: Youtube

Tech: Big Data Visualization
  Computational Literature
  LED Display

Shakespeare Machine is a large-scale multimedia sculpture suspended from the ceiling of New York’s Public Theatre, the only theatre dedicated to Shakespeare’s works in the city. The sculpture also serves as the space’s chandelier and leaves a modern taste for the enduring works of the eminent figure.

The work features 37 LED screens on which words extracted from Shakespeare’s original masterpieces were displayed, creating an unfolding kaleidoscope of language in motion.

*The project was built in collaboration with Parallel Development.
